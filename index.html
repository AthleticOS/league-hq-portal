Link Your Google Sheets

Because your platform is hosted inside Google Workspace, you can turn your Google Spreadsheet into a read-only public database using Google Apps Script. Paste your deployed Web App URL in the box below to swap the mock demo data with your real-time league records!

Requires a deployed Web App script

Setup Guide: Deploying Your Read-Only Google Apps Script API

1 Open Google Spreadsheet Script Editor

Open the Google Sheet that houses your league database. Click on Extensions > Apps Script from the top menu.

2 Paste the API Code

Delete any auto-generated code and paste the complete, read-only API script below. 

function doGet() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const tables = ["Messages", "Fixtures", "Sports", "Schools", "Teams", "Roster", "Divisions"];
  const database = {};

  tables.forEach(tableName => {
    const sheet = ss.getSheetByName(tableName);
    if (!sheet) {
      database[tableName] = [];
      return;
    }
    const values = sheet.getDataRange().getValues();
    if (values.length <= 1) {
      database[tableName] = [];
      return;
    }
    const headers = values[0];
    const rows = [];
    for (let i = 1; i < values.length; i++) {
      const rowObj = {};
      headers.forEach((header, index) => {
        let val = values[i][index];
        // Clean Date Objects into readable string formats
        if (val instanceof Date) {
          val = val.toISOString().split('T')[0];
        }
        rowObj[header] = val;
      });
      rows.push(rowObj);
    }
    database[tableName] = rows;
  });

  return ContentService.createTextOutput(JSON.stringify(database))
    .setMimeType(ContentService.MimeType.JSON);
}
3 Deploy as a Web App

1. In the upper right corner, click Deploy > New Deployment.
2. Click the gear icon next to "Select type" and choose Web app.
3. Set "Execute as" to Me (your email address).
4. Set "Who has access" to Anyone (this allows the read-only portal to view schedules without asking viewers to log into Google).
5. Click Deploy and copy the generated Web App URL! Paste that URL into the input field above.
